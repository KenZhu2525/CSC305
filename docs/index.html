About Campus Canvas

Campus Canvas is a community-driven mobile app with a unique twist: users must physically visit specific real-life locations to place virtual pixels based on their real-world location. This app draws inspiration from Reddit, particularly r/Place, where a vast canvas allows users to add a single pixel every few minutes.
